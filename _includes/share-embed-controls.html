{%- comment -%} Share Embed Controls The embed-size controls row inside the share panel's embed section — the preset dropdown (Canvas, Moodle, WordPress, and so on), the width and height inputs, and the copy button. Shared by both branches of share-panel.html (story page and homepage) so the two cannot drift apart. The element ids (#embed-preset-select, #embed-width-input, #embed-height-input, #embed-copy-code-btn) are read by assets/js/share-panel.js; only one branch renders per page, so each id appears once in the document. Jekyll include scope does not reliably inherit the parent layout's `lang` assign (see _includes/object-grid-item.html for the same note), so `lang` is passed in explicitly rather than relied on from the including template's scope. Parameters: lang - the active language pack (site.data.languages[...]) Version: v1.6.0 {%- endcomment -%}
