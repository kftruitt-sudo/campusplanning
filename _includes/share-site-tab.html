{%- comment -%} Share Site Tab The "this whole site" tab pane of the share panel — a read-only link input (filled in at runtime by the share JavaScript) with a copy button. Shared by both branches of share-panel.html (story page and homepage) so the two cannot drift apart. The element ids (#content-site, #share-site-url-input, #share-copy-site-btn) are read by assets/js/share-panel.js; only one branch renders per page, so each id appears once in the document. Jekyll include scope does not reliably inherit the parent layout's `lang` assign (see _includes/object-grid-item.html for the same note), so `lang` is passed in explicitly rather than relied on from the including template's scope. Parameters: lang - the active language pack (site.data.languages[...]) Version: v1.6.0 {%- endcomment -%}
