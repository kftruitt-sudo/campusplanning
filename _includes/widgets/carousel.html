{# Carousel Widget Renders a swipeable slideshow of images — a Bootstrap carousel — with clickable indicator dots and previous/next controls, advancing only on reader action (data-bs-ride is false, so it never auto-plays). Produced by Telar's widget build step, which supplies: widget_id (a unique slug scoping the carousel's id and its control targets so multiple carousels coexist on a page); items (the slides, each with a pre-resolved src, alt, and optional caption and credit, the latter two rendered below the image and passed through "safe" so authored markup survives); and size_class (an optional preset that adds a carousel-size-* modifier, ignored when "default"). The template does no URL logic itself — parse_carousel_widget() in scripts/telar/widgets.py resolves each item's src before this template ever sees it: absolute http(s) URLs pass through unchanged, bare filenames are joined to the site's asset path, so the same widget works for both externally hosted and site-local images. Control labels come from the language pack via the build step: slide_label (whose number token is substituted per slide below), prev_label, and next_label. #}
